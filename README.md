# Jaideep Enterprises – Modern Website Redesign

A premium, modern, and responsive website built for **Jaideep Enterprises**, a manufacturer and exporter of precision brass components with an additional business line in CCTV components.

This project focuses on **clean industrial UI, strong trust-building design, and conversion-focused structure** — suitable for a real-world B2B client.

---

## Project Overview

This website was redesigned from a basic static HTML site into a **modern, high-end frontend experience** with:

* Strong visual hierarchy
* Professional industrial branding
* Smooth animations
* Clear product showcase
* Lead generation focus

---

## Features

### Main Website (Brass Components)

* Premium hero section with strong messaging
* Trust indicators (experience, export quality, clients)
* Product showcase with interactive cards
* "Why Choose Us" section
* Manufacturing capability highlights
* Contact form + WhatsApp CTA
* Smooth scroll + animations (Framer Motion)

---

### CCTV Components Page

* Dedicated page under same domain (`/cctv-components`)
* Red, black, and white theme (brand-specific)
* Product categories (connectors, mounts, cables, accessories)
* High-contrast UI for strong visual identity
* Clear CTA for inquiries and quotes

---

## Design Philosophy

This project follows a **B2B industrial design approach**:

* Clean layout with ample whitespace
* Trust-first UI (not flashy startup style)
* Strong typography and hierarchy
* Conversion-focused sections
* Subtle, premium animations

---

## Tech Stack

* **React (Vite)**
* **Tailwind CSS**
* **Framer Motion** (animations)
* **Lucide Icons**

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/JiteshSuvarnaK/JAIDEEP-ENTREPRISES.git
cd new1
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```

## Deployment

This project can be easily deployed using:

### Vercel (Recommended)

```bash
npm run build
npx vercel
```

### Netlify

* Build command: `npm run build`
* Publish directory: `dist`
