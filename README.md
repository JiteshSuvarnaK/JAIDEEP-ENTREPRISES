# Jaideep Enterprises – Modern Website Redesign

A modern, responsive website built for Jaideep Enterprises, a manufacturer and exporter of precision brass components with an additional business line in CCTV components.

This project focuses on clean industrial UI, strong trust-building design, and a conversion-oriented structure suitable for a real-world B2B business.

---

## Project Overview

This website was redesigned from a basic static HTML site into a modern, high-quality frontend experience with:

* Strong visual hierarchy
* Professional industrial branding
* Smooth transitions and refined UI
* Clear product presentation
* Lead generation focus

---

## Features

### Main Website (Brass Components)

* Structured hero section with strong messaging
* Trust indicators (experience, export quality, manufacturing capability)
* Product showcase with clean layout
* "Why Choose Us" section for credibility
* Manufacturing capability highlights
* Contact section with clear call-to-actions
* Smooth scrolling and visual consistency

---

### CCTV Components Page

* Dedicated page under the same domain (`/cctv-components`)
* Red, black, and white theme aligned with product branding
* Product categories (connectors, mounts, cables, accessories)
* High-contrast layout for clear differentiation
* Direct inquiry and quote-focused CTAs

---

## Design Philosophy

This project follows a B2B industrial design approach:

* Clean layout with proper spacing
* Trust-first UI instead of flashy visuals
* Strong typography and structured hierarchy
* Conversion-focused sections
* Consistent and professional styling

---

## Tech Stack

## Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/JiteshSuvarnaK/JAIDEEP-ENTREPRISES.git
cd new1
```

### 2. Open the project

You can run the project directly by opening `index.html` in your browser.

Alternatively, use a local server:

```bash
npx serve
```
